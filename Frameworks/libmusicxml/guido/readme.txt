-----------------------------------------------------------
	            MusicXML to Guido converter
-----------------------------------------------------------
 You'll find below the list of elements supported by 
 the converter, followed by the list of unsupported 
 elements.
 Note that these lists may be slightly inaccurate since 
 some elements are not handled using the visitor pattern
 but queried directly from the container (e.g. this is the 
 case for dynamics).
 Note also that a container present in the list of supported
 elements doesn't implies that all its enclosed elements are
 supported.
 Have also a look at the 'todo' comments inside the code.
-----------------------------------------------------------
